Melanoma Classifier
Prediction on Melanoma skin lessions with 82% accuracy and 82% recall

Getting Started

Prerequisites
Python 3.8

pip install -r requirements.txt

Getting the code
git clone https://github.com/carlosdanielgt/MelanomaClassifier


Running the code

MLP Classifier
To run the Multilayer Perceptron (MLP) classifier:
python MLPClassifier.py

1D 10-Fold CNN
To execute the 1-dimensional CNN:
python 1D_10fold_CNN.py

CNN Classifier
To run the Convolutional Neural Network (CNN) classifier:
python CNNClassifier.py
